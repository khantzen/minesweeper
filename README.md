### MineSweeper

Simple implementation of a mine sweeper game

#### Pre-requisite
Having both package installed

- Gradle 4.10.1+
- JDK 8+

#### Getting Started

##### How to run
```shell
git clone https://github.com/khantzen/minesweeper.git
cd minesweeper
gradle run --args='142857'
```

##### How to build
```shell
gradle clean build
```
